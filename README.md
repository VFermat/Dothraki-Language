# Dothraki Language

A computer language based on Dothraki Idiom, from the TV Show and Book Series Game of Thrones

**EBNF**: [Link](EBNF.md)

## Variable Types

- ASE -> String
- ATO -> Integer
- SOM -> Void
- TAWAK -> Boolean
- NAQIS -> Float

## Lexical Features

- ATAKI -> Start of command
- NAKHO -> End of statement/line
- FIN -> If
- ESHNA -> Else if
- NAKHAAN -> Else
- HAEI -> While
- EZAT -> Return
